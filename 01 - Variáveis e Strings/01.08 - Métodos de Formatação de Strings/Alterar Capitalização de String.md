# Como você pode alterar a capitalização de uma string?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

Use toUpperCase para maiúsculas e toLowerCase para minúsculas. Como strings são imutáveis, o método retorna uma nova string.

## Exemplo em JavaScript

```js
const nome = "aNa";
console.log(nome.toUpperCase());
console.log(nome.toLowerCase());
```
