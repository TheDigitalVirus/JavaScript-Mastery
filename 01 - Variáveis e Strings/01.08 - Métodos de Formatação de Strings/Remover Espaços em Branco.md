# Como você pode remover espaços em branco de uma string?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

Use trim para remover espaços no início e no fim, trimStart para o início e trimEnd para o fim.

## Exemplo em JavaScript

```js
const entrada = "  JavaScript  ";
console.log(entrada.trim());
```
