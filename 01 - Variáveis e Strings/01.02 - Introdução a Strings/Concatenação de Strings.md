# O que é concatenação de strings e como você pode concatenar strings com variáveis?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

Concatenação é unir textos. Ela pode ser feita com o operador + ou com template literals, que geralmente deixam o código mais legível.

## Exemplo em JavaScript

```js
const nome = "Lia";
const frase = "Olá, " + nome + "!";
const moderna = `Olá, ${nome}!`;
```
