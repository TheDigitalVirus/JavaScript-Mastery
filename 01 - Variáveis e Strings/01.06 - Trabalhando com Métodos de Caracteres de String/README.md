# Trabalhando com Métodos de Caracteres de String

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

Esta seção agrupa métodos e técnicas para acessar, comparar e manipular caracteres em strings.

## Exemplo em JavaScript

```js
const texto = "JavaScript";
console.log(texto.charAt(0));
console.log(texto.at(-1));
```
