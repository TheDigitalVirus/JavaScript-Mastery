# Como Você Pode Testar se uma String Contém uma Substring?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

Use includes para verificar se uma string contém outra. O resultado é booleano.

## Exemplo em JavaScript

```js
const email = "ana@example.com";
console.log(email.includes("@"));
```
