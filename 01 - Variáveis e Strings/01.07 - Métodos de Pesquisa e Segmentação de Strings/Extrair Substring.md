# Como você pode extrair uma substring de uma string?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

Use slice para extrair parte de uma string. Ele recebe índice inicial e, opcionalmente, índice final exclusivo.

## Exemplo em JavaScript

```js
const texto = "JavaScript";
console.log(texto.slice(0, 4));
console.log(texto.slice(4));
```
