# O que é o método prompt() e como ele funciona?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

prompt é uma função do navegador que abre uma caixa de entrada para o usuário e retorna o texto digitado ou null se for cancelada.

## Exemplo em JavaScript

```js
const nome = prompt("Qual é seu nome?");
console.log(`Olá, ${nome}!`);
```
