# O que são Template Literals e o que é String Interpolation?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

Template literals usam crases e permitem interpolar expressões com ${}. Também facilitam strings em múltiplas linhas.

## Exemplo em JavaScript

```js
const nome = "João";
const pontos = 15;
console.log(`Olá, ${nome}. Você tem ${pontos} pontos.`);
```
