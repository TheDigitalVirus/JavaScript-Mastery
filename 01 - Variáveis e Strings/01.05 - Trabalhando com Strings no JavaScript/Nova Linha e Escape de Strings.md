# Como Você Cria uma Nova Linha em Strings e Escapa Strings?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

Use sequências de escape como `\n` para nova linha, `\\` para barra invertida e aspas escapadas quando necessário.

## Exemplo em JavaScript

```js
const texto = "Linha 1\nLinha 2";
const citacao = "Ela disse: \"Olá!\"";
```
