# Como você pode encontrar a posição de uma substring em uma string?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

Use indexOf para encontrar a primeira posição de uma substring. Se não encontrar, o retorno é -1.

## Exemplo em JavaScript

```js
const frase = "Aprendendo JavaScript";
console.log(frase.indexOf("JavaScript"));
console.log(frase.indexOf("Python"));
```
