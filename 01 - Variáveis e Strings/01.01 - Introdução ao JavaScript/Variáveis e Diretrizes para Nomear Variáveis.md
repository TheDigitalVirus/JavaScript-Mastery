# O que são Variáveis e quais são as Diretrizes para Nomear Variáveis em JavaScript?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

Variáveis são nomes que guardam valores para reutilização. Bons nomes devem ser claros, específicos, em camelCase e indicar intenção.

## Exemplo em JavaScript

```js
const nomeUsuario = "Marina";
let pontosAtuais = 10;
pontosAtuais = pontosAtuais + 5;
```
