# O que é JavaScript e como ele funciona com HTML e CSS?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

JavaScript é a linguagem que adiciona comportamento e interatividade às páginas web. HTML estrutura o conteúdo, CSS define a aparência e JavaScript controla ações, eventos, validações e mudanças dinâmicas na interface.

## Exemplo em JavaScript

```js
const botao = document.querySelector("button");
botao.addEventListener("click", () => {
  console.log("Usuário clicou!");
});
```
