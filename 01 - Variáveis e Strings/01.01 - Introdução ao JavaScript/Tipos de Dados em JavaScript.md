# O que é um Tipo de Dado e quais são os Diferentes Tipos de Dados em JavaScript?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

Um tipo de dado descreve a categoria de um valor. JavaScript possui tipos primitivos como string, number, boolean, undefined, null, symbol e bigint, além de objetos.

## Exemplo em JavaScript

```js
const nome = "Ana";
const idade = 25;
const ativo = true;
console.log(typeof nome, typeof idade, typeof ativo);
```
