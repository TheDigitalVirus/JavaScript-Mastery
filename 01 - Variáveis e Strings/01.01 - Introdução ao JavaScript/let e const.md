# Como `let` e `const` Funcionam Diferentemente Quando Se Trata de Declaração, Atribuição e Reatribuição de Variáveis?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

Use const quando a variável não será reatribuída e let quando o valor precisa mudar. Ambos possuem escopo de bloco.

## Exemplo em JavaScript

```js
const linguagem = "JavaScript";
let nivel = 1;
nivel = 2;
```
