# O que é Tipagem Dinâmica em JavaScript e Como Ela Difere das Linguagens de Tipagem Estática?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

JavaScript é dinamicamente tipado: o tipo pertence ao valor, não à variável. A mesma variável com let pode receber valores de tipos diferentes.

## Exemplo em JavaScript

```js
let valor = "10";
valor = 10;
console.log(typeof valor);
```
