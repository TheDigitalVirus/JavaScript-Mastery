# Como o operador typeof funciona e qual é o bug typeof null em JavaScript?

## Objetivo

Entender o conceito, reconhecer quando usá-lo e praticar com pequenos exemplos em JavaScript.

## Explicação simples

typeof retorna uma string com o tipo de um valor. O caso typeof null retorna object por um comportamento histórico da linguagem.

## Exemplo em JavaScript

```js
console.log(typeof "texto");
console.log(typeof 42);
console.log(typeof null); // "object"
```
