# Glossário

## JavaScript

Linguagem de programação usada para adicionar comportamento a páginas web, criar aplicações, automatizar tarefas e trabalhar no front-end e no back-end.

## Variável

Nome que guarda uma referência para um valor. Relacionado: [[01 - Variáveis e Strings/01.01 - Introdução ao JavaScript/Variáveis e Diretrizes para Nomear Variáveis|Variáveis e Diretrizes]].

## String

Tipo de dado usado para representar texto. Relacionado: [[01 - Variáveis e Strings/01.02 - Introdução a Strings/O que é uma String|O que é uma String]].

## Tipo de dado

Categoria de valor, como `string`, `number`, `boolean`, `undefined`, `null`, `symbol`, `bigint` e `object`.

## Template literal

Forma moderna de criar strings com crase e interpolação: `` `Olá, ${nome}` ``.

## Imutabilidade

Característica de valores que não podem ser alterados diretamente. Strings em JavaScript são imutáveis.
