# Documentação

## Fontes recomendadas

- MDN Web Docs: referência principal para JavaScript, APIs Web, HTML e CSS.
- JavaScript.info: curso textual progressivo com muitos exemplos.
- ECMAScript Specification: referência formal da linguagem.

## Como usar documentação

1. Pesquise o método ou conceito.
2. Leia a definição.
3. Rode um exemplo pequeno.
4. Crie um resumo em uma nota própria.
5. Adicione links para notas relacionadas.
