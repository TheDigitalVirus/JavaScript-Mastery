# JavaScript - Tipos de Dados

#flashcards #flashcards/javascript/tipos-de-dados #javascript #tipos-de-dados

O que `typeof` retorna?::Uma string indicando o tipo do valor avaliado.

Qual é o resultado de `typeof null`?::`"object"`, por um comportamento histórico da linguagem.

O que é tipagem dinâmica?::O tipo pertence ao valor, e variáveis podem receber valores de tipos diferentes.

Cite três tipos primitivos de JavaScript.::`string`, `number` e `boolean`.
