# JavaScript - Variáveis

#flashcards #flashcards/javascript/variaveis #javascript #variaveis

O que é uma variável?:: Um nome que guarda uma referência para um valor.

Quando usar `const`?:: Quando a variável não precisa ser reatribuída.

Quando usar `let`?:: Quando o valor da variável precisa mudar durante o programa.

O que é camelCase?:: Convenção de escrita em que a primeira palavra começa minúscula e as próximas começam com maiúscula, como `nomeUsuario`.
