# JavaScript - Strings

#flashcard #javascript #strings

O que é uma string em JavaScript?:: Um tipo de dado usado para representar texto.

Strings em JavaScript são mutáveis ou imutáveis?:: Imutáveis.

Como acessar o primeiro caractere de `nome`?:: `nome[0]`.

Para que serve `includes()`?:: Para verificar se uma string contém uma substring.

Para que serve `trim()`?:: Para remover espaços em branco no início e no fim da string.
