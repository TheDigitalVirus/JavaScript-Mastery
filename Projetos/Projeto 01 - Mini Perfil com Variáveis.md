# Projeto 01 - Mini Perfil com Variáveis

## Objetivo

Criar um mini perfil no console usando variáveis, tipos de dados e template literals.

## Requisitos

- [ ] Criar `nome`, `idade`, `cidade` e `estudandoJavaScript`.
- [ ] Exibir uma frase usando template literal.
- [ ] Usar `typeof` para verificar pelo menos dois valores.

## Exemplo esperado

```js
const nome = "Ana";
const idade = 22;
const cidade = "Porto";
const estudandoJavaScript = true;

console.log(`${nome}, ${idade} anos, mora em ${cidade}.`);
console.log(typeof nome);
console.log(typeof estudandoJavaScript);
```
